## 📖 À propos

Hub RL est un dashboard interactif que j'ai développé pendant ma période de révision pour les examens. Plutôt que de simplement relire mes cours de Reinforcement Learning, j'ai décidé de les transformer en une plateforme pédagogique complète et interactive.

**Mon objectif ?** 
- ✅ Mémoriser mieux en construisant
- ✅ Développer mes compétences en développement web
- ✅ Créer une ressource utile pour la communauté

---

## 🎯 Fonctionnalités

### 📚 Contenu pédagogique
- **5 chapitres complets** couvrant tous les aspects du RL
- **Schémas interactifs** pour chaque concept
- **Explications détaillées** avec exemples concrets

### 🎮 Interactivité
- **Quiz interactifs** à la fin de chaque chapitre (3 questions)
- **Barre de recherche** intelligente pour trouver rapidement un concept
- **Animations fluides** avec Framer Motion
- **Thème clair/sombre** (toggle)

### 🎨 Design
- Interface moderne et responsive
- Animations et transitions soignées
- Navigation intuitive entre les chapitres

---

## 📖 Contenu des chapitres

| Chapitre | Titre | Sujets couverts |
|----------|-------|-----------------|
| **I** | Introduction | Définition du RL, différences avec supervisé/non-supervisé, applications réelles |
| **II** | Fondamentaux | Agent, Environnement, État, Action, Récompense, MDP, Policy, Value, Model |
| **III** | Algorithmes | Q-Learning, SARSA, PPO, Dyna-Q |
| **IV** | Deep RL | DQN, Experience Replay, Target Network, Actor-Critic, Policy Networks |
| **V** | Frameworks | TensorFlow, PyTorch, applications industrielles, RLOps |

## 🚀 Installation locale

```bash
# Cloner le dépôt
git clone https://github.com/gueddourikhawla-lgtm/hub-RL.git

# Accéder au dossier
cd hub-RL

# Installer les dépendances
npm install

# Lancer en mode développement
npm run dev

# Build pour production
npm run build
